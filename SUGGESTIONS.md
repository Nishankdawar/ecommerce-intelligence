## Inventory Intelligence

### S1 — Download 30 days of Inventory Ledger (not 3 days)

**Current state:** `inv_ledger.csv` covers only 3 days (Apr 29 – May 1, 2026).

**Problem:**
A 3-day window is too narrow to distinguish a slow-moving SKU from a dead one. Amazon doesn't ship every SKU every single day — a SKU that sells 5 units every 10 days will show zero shipments in a 3-day window and get wrongly classified as dead inventory.

**Real example from current data:**
```
Current result:  1,299 out of 1,471 SKUs (88.3%) flagged as "dead inventory"
Estimated cost:  ₹18.44L/month shown as dead storage cost

Reality check:
  OM-ROUND-SELENITE-PLATE-3IN  → 14 units in stock, 0 shipments in 3 days → flagged DEAD
  HANUMAN-JI-GADA-ST-3         → 50 units in stock, 0 shipments in 3 days → flagged DEAD
  3IN-S.STEEL-STUD-SET-9       → 49 units in stock, 0 shipments in 3 days → flagged DEAD

These SKUs likely sell a few units per week — they appear dead only because
we happened to not have a shipment in those specific 3 days.
```

**What changes with 30 days:**
```
30-day window example for a SKU selling 10 units/month:
  3-day window:   0 shipments → classified DEAD (wrong)
  30-day window:  10 shipments → velocity = 0.33/day → classified OK (correct)
                  Days remaining = stock / 0.33 → accurate stockout prediction
```

**Recommendation:**
```
Reports → Fulfillment → Inventory → Inventory Ledger → Date range: last 30 days
```
With 30 days: velocity is accurate, dead inventory is meaningful, stockout dates are reliable.

## Data Coverage

### S2 — April MTR B2C is missing

**Current state:** MTR B2C available: Jan, Feb, Mar 2026. April is missing.

**Problem:**
The Business Report covers Jan–Apr (total revenue ₹1.12Cr) but the daily trend chart can only show Jan–Mar because MTR B2C is the source for day-by-day revenue breakdown. This creates a visible inconsistency on the Dashboard.

**Real example from current data:**
```
Dashboard summary cards show:   ₹1.12Cr total revenue (Jan–Apr, from Business Report)
Daily trend chart shows:        Jan–Mar only (₹75.8L across 3 months)

The gap:
  Jan 2026:  ₹22.24L  ✓ visible in chart
  Feb 2026:  ₹26.09L  ✓ visible in chart
  Mar 2026:  ₹27.47L  ✓ visible in chart
  Apr 2026:  ₹35.88L  ✗ missing from chart — user sees a blank/cut-off trend

A user looking at the dashboard sees ₹1.12Cr total but the trend line stops at
March 31, making it look like April had zero revenue — which is misleading.
```

**Recommendation:**
Download and upload April MTR B2C (generated by Amazon in early May):
```
Reports → Tax Document Library → GST Reports → MTR → B2C → April 2026
```
Once uploaded, the trend chart will extend to April 30 and match the summary cards.
